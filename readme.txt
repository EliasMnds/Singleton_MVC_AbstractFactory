O trabalho consiste em desenvolver um sistema que possui duas entidades.

O sistema deve permitir a inserção, alteração,
exclusão dos dados das entidades. Também deve ser possível fazer a busca a partir do id e de um outro atributo a
escolha. 
Deve ser possível realizar essas operações de forma independente para cada entidade. 
Além disso, para
a entidade principal, deve ser possível adicionar e remover objetos da outra entidade por meio da busca por id ou
nome de objetos pré-cadastrados.
Todas as classes concretas de persistência devem ser Singletons.
Todos os objetos das camadas do MVC devem ser criados com o uso do padrão Abstract Factory.
O modelo de classes deve ser atualizado para refletir o estado atual do sistema.

Trabalho desenvolvido na disciplina de Introdução a Engenharia de Software na Universidade Federal de São João Del Rei
